#--------------------------------------------------------------------------#
############################## Sketchers Code ##############################
#--------------------------------------------------------------------------#

This directory holds the source code to the Sketchers projects, currently
dating between 2006 and 2011.

--- Subdirectories ---

Build: Stuff for the finished software. Right now, it just holds the
	Windows Installer project.

FrontEnd: user interfaces and power point interfaces

Recognition: Single- and multi-stroke recognition algorithms,
	clustering/grouping code, circuit recognition code, truth-table 
	recognition code, and support code that is only used for recognition.

Util: utilities used in recognition code or stand-alone tools such as
	analysis and testing programs

-----------------------

The solution file you'll probably want to work with is CircuitSimulatorUI.sln 
inside FrontEnd/WPFCircuitSimulatorUI.
